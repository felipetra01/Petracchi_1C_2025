◆ mostrarNumero()

int8_t mostrarNumero ( uint32_t numero,
uint8_t digitos,
const gpioConf_t * bcd_config,
const gpioConf_t * lcd_config )

Muestra un número en un display BCD utilizando GPIOs.

Esta función toma un número decimal, lo convierte a BCD (Binary-Coded Decimal) y lo muestra en un display LCD. El display se controla mediante GPIOs configurados para los dígitos BCD y para la selección de dígitos en el LCD.

Parámetros
numeroNúmero decimal (0-999) de tipo uint32_t a mostrar
digitosCantidad de dígitos a mostrar (1-3)
bcd_configPuntero a la configuración de GPIOs para selección de dígitos
lcd_configPuntero a la configuración de GPIOs para los segmentos del LCD
Devuelve
int8_t Retorna 0 si la operación fue exitosa, -1 si hubo un error
Nota
La función verifica que el número de dígitos sea válido antes de procesar
En modo DEBUG, la función imprime información detallada del proceso
